HTML Tables

In HTML tables are created using the <table> tag.The table is structured with rows (<tr>) and cells, which can be either header cells (<th>) or data cells (<td>).A ttributes like border, rowspan and colspan can be used to enhance the table's appearance and structure

Basic Table

Header1 Header2 Header3
Row1,Cell1 Row1,Cell2 Row1,Cell3
Row2,Cell1 Row2,Cell2 Row2,Cell3

Table with rowspan

The rowspan attribute allows a cell to span multiple rows

Header1 Header2 Header3
Rowspan 2 rows Row1,Cell2 Row1,Cell3
Row2,Cell2 Row2,Cell3

Table with colspan

The colspan attribute allows a cell to span multiple columns

Header1 Header2 Header3
Row1,Cell1 colspan 2 columns Row1,Cell3
Row2,Cell1 Row2,Cell2 Row2,Cell3
